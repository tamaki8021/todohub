## 成果物内容

基本的なTodoアプリにプラスαでGitHubの草機能、そしてTodoを評価できるようにスターrerating機能をつけて、
オリジナルのTodoアプリとして作成しました。


## つけた機能

TODO機能　CRUD
新規登録、ログイン、ログアウト、パスワードリセット
簡単に使ってもらえるようにログインの場合に簡単ログイン機能をつけました。
Todoの成果を可視化するためのヒートマップ（GitHubの草機能）
